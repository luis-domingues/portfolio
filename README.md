This is a [Tailwindcss](https://tailwindcss.com/) project.

## Getting Started
First, you need to clone this repository:

```bash
git clone https://github.com/luis-domingues/portfolio
```
After, navigate to the project directory and start the Tailwind CLI biuld process:

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
If in doubt, consult the [Tailwind documentation](https://tailwindcss.com/docs/installation).
